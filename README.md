# Habitat — AI Songwriting Engine

> *"Don't generate a song. Excavate one."*

Habitat is a full-stack AI songwriting system that guides users through identity discovery
and transforms that identity into structured music using a custom creative engine.

This is **not** a simple lyric generator. It is a structured intelligence pipeline.

---

## The Pipeline

```
User Identity → Persona → Message → Structure → AI Song Generation
```

Every song generated by Habitat is the product of:
- **Identity interrogation** — confrontational questions that surface emotional truth
- **Persona modeling** — archetype, voice, tone, energy, perspective
- **Message extraction** — core message + sub-themes derived from answers
- **Structure planning** — dynamically generated section order based on conflict type
- **Stylistic mapping** — rhyme scheme, flow, language blend, imagery style
- **Section-by-section AI generation** — no freeform generation, ever

---

## Features

- 🎤 7-step progressive identity questionnaire (confrontational, not a form)
- 👤 Persona visualization (archetype, voice, emotional profile)
- 🏗 Dynamic song structure planning
- 🌍 Multilingual support — English, Kiswahili, Sheng blending
- ⚡ Real-time section-by-section generation with live lyrics stream
- 📋 Copy & download generated songs
- 🔌 Provider abstraction — works with Claude (Anthropic) or GPT-4o (OpenAI)

## Quick Start

### Prerequisites
- Node.js 18+
- An Anthropic API key (`sk-ant-...`) or OpenAI API key

### 1. Clone
```bash
git clone https://github.com/jaguar999paw-droid/sci-songwriting-engine.git
cd sci-songwriting-engine
```

### 2. Install Backend
```bash
cd backend
npm install
```

### 3. Install Frontend
```bash
cd ../frontend
npm install
```

### 4. Configure Environment
```bash
cp .env.example backend/.env
# Edit backend/.env — set PORT=3001 if needed
```

### 5. Run
In terminal 1:
```bash
cd backend && npm start
```
In terminal 2:
```bash
cd frontend && npm run dev
```

Open **http://localhost:3000** — enter your API key in the UI and begin.

## Project Structure

```
sci-songwriting-engine/
  /frontend               React UI (Vite)
    /src
      /pages              Landing, Questionnaire, PersonaReview, Generator, SongDisplay
      /components         PersonaCard, StructurePlan, ProgressBar
      /styles             Global CSS variables and animations
  /backend                Express API server
  /engine                 Rule-based creative intelligence (pure JS, no AI)
    identityParser.js     Emotion, conflict, trait detection
    personaBuilder.js     Persona object construction
    messageExtractor.js   Core message + sub-theme derivation
    structurePlanner.js   Dynamic song structure planning
    styleMapper.js        Rhyme, flow, language, imagery mapping
  /ai                     AI integration layer
    promptBuilder.js      Structured prompt construction
    generator.js          Claude/OpenAI API calls, section-by-section
  /docs                   Architecture, ideology, contribution guides
```

## How to Extend the Engine

| Goal | File to Edit |
|---|---|
| Add a persona archetype | `engine/personaBuilder.js` → `ARCHETYPE_MAP` |
| Add a language | `engine/styleMapper.js` → `LANGUAGE_BLEND_INSTRUCTIONS` |
| Add a structure template | `engine/structurePlanner.js` → `STRUCTURE_TEMPLATES` |
| Add a questionnaire question | `frontend/src/pages/Questionnaire.jsx` → `QUESTIONS` |
| Add an AI provider | `ai/generator.js` → add `callProvider()` function |

See [docs/contribution.md](docs/contribution.md) for detailed guides.

## Philosophy

Habitat is built on the songwriting methodology documented in `SONG_QUESTIONAIRE.md`,
which uses WH-questions (Who, What, When, Where, Why, How) across 10 categories
— persona, message, emotion, rhythm, structure, cultural sensitivity — as a
systematic framework for artistic excavation.

See [docs/ideology.md](docs/ideology.md) for the full philosophy.
